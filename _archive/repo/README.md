# SME Brain Dump

SME Brain Dump is an AI-powered interview tool for capturing subject matter expert knowledge before a role transition and turning it into a usable handoff package.

## What ships
- Stub interview flow with branching prompts
- Example scenario: Sally Chen, Data Pipeline Owner
- Four generated outputs, Successor Brief, FAQ, Decision Log, Runbook
- Live-mode settings path for LLM and ElevenLabs keys
- Landing page for GitHub Pages

## Run locally
Open `index.html` in a browser, or serve the folder with any static server.

## Files
- `app.html` app experience
- `stub.html` stub mode shortcut
- `live.html` live mode shortcut
- `example.html` example scenario shortcut
- `index.html` landing page

## Notes
- Stub mode works with no configuration.
- Live mode is architected as a clean extension path.
